# 🚀 Web Automation Suite

## 🎯 Objective
The main objective of this project is to **learn and practice core concepts of Selenium Automation** through hands-on implementation.  
This suite is designed to strengthen automation skills by covering real-world scenarios and essential Selenium features.

---

## 💡 Areas Covered

- Action Class & Robot Class (Mouse hover, Double click, Right click, Drag & Drop, Keyboard actions)  
- Alerts Handling (Simple, Confirmation, Prompt)  
- Assertions (Hard & Soft Assertions)  
- Cookies Handling (Add, Delete, Retrieve)  
- Dropdown Handling (Single & Multi-select)  
- File Handling (Upload & Download)  
- Frames Handling (Single & Nested Frames)  
- JavaScript Executor (Scrolling & Element interaction)  
- Locators (ID, Name, Tag Name, Link Text, CSS Selector, XPath)  
- Network Handling (Broken links & Response validation)  
- Parameterization (Using TestNG XML)  
- Screenshots (Full page & Web element)  
- Web Tables (Static & Dynamic)  
- TestNG Annotations (Before/After Suite, Test, Class, Method)  
- TestNG Features (Priority, Groups, DependsOnMethods, InvocationCount)  
- Waits (Implicit, Explicit, Fluent)  
- Window Handling (Single & Multiple windows)  

---

## 📌 Learning Outcome

- Built strong fundamentals in Selenium automation  
- Gained hands-on experience with real-time testing scenarios  
- Improved debugging and problem-solving skills  
- Learned best practices for writing maintainable test scripts
